# Poker Scanner — локальный Python агент

Захватывает экран, распознаёт карты и суммы, отправляет GTO-анализ в Telegram.  
Работает на Windows. Протестировано с **Ton Poker** (десктоп и мобильный браузер).

---

## Быстрый старт

### 1. Установи зависимости

```bash
pip install -r requirements.txt
```

> При первом запуске EasyOCR скачивает модель (~100 МБ).

---

### 2. Создай config.json

Скопируй `config.example.json` → `config.json` и заполни:

```json
{
  "server_url": "https://ТВОЙ_ДОМЕН.replit.dev/api/python/scan",
  "position": "BTN",
  "players": 6
}
```

---

### 3. Загрузи готовый пресет (Ton Poker)

```bash
python load_preset.py
```

```
=== Доступные пресеты ===

  [1] Ton Poker — Desktop (Windows app/browser, 6-max, ландшафт)
  [2] Ton Poker — 6-max, портрет, мобильный браузер

Введи номер пресета: 1
```

Пресет скопирует готовые координаты карт и банка в `config.json`.

> ⚠️ Пресет даёт **стартовые** координаты. Запусти `calibrate.py` чтобы точно
> подстроить под свой экран, или сразу проверь через `poker_scanner.py`.

---

### 4. Точная калибровка (если пресет не совпал)

```bash
python calibrate.py
```

**Фаза 1 — Карты** (клики по центру):

| # | Что кликать |
|---|-------------|
| 1–2 | Свои карты (левая, правая) |
| 3–5 | Борд — флоп (3 карты) |
| 6 | Борд — тёрн *(Q = пропустить)* |
| 7 | Борд — ривер *(Q = пропустить)* |

Нажми **S** → Фаза 2.

**Фаза 2 — Деньги** (зажать ЛКМ и обвести):

| # | Что обводить |
|---|-------------|
| 1 | Цифра банка (жёлтый кружок над бордом) |
| 2 | Цифра колла/ставки *(Q = пропустить)* |

**S** = сохранить.

> Разверни окно игры на весь экран ДО калибровки. Изменил размер — перекалибруй.

---

### 5. Собери шаблоны карт (один раз, ~2-3 раздачи)

```bash
python collect_templates.py
```

Просто играй — скрипт автоматически сохраняет шаблоны карт.  
Прогресс виден в окошке и в терминале:

```
[████████████████░░░░░░░░░░░░░░░░░░░░░░░░] 16/52
  💾 10d  (уверенность 0.94)
  💾 Jh   (уверенность 0.91)
```

Когда все 52 — скрипт останавливается сам. Шаблоны хранятся в `templates/` навсегда.

---

### 6. Запусти сканер

```bash
poker_scanner.py
```

При старте показывает статус:

```
🃏 Template matching: загружено 52/52 шаблонов
💰 Банк: OCR
💰 Колл: OCR
Сканирование запущено (5 FPS). Ctrl+C для остановки.
```

В терминале по мере игры:

```
→ CALL   Win 24%  |  Ah Kd | 2c 7h Qd  банк=120  колл=30
→ FOLD   Win 8%   |  7c 2h |
→ RAISE  Win 67%  |  As Ad | Kc 7h 2d  банк=45
```

В Telegram одновременно:

```
🟢 CALL
A♥ K♦  ·  2♣ 7♥ Q♦
Win 24%
```

## Поддерживаемые пресеты

| Пресет | Ориентация | card_height_pct |
|--------|-----------|----------------|
| Ton Poker Desktop (Windows app) | Ландшафт ~1040×585 | 10 |
| Ton Poker Mobile (браузер) | Портрет ~468×847 | 9 |

Если у тебя другое разрешение — запусти `calibrate.py` для точной настройки.

---

## Как работает распознавание карт

| Режим | Когда | Скорость | Точность |
|-------|-------|----------|----------|
| **Template matching** | Шаблон для карты есть | ~2–5 мс | ~99% |
| **EasyOCR (fallback)** | Шаблон не найден | ~100–300 мс | ~85% |

### Определение масти

Двухфазная логика — работает для **любого** рума:

- **Фаза 1** (насыщенные пиксели): цветные румы (синие/зелёные масти) + красные ♥♦ на белом фоне
- **Фаза 2** (тёмные пиксели в центре): чёрные ♣♠ на белом фоне (Ton Poker, PokerStars и др.)
  - ♠ (пика): острый верх → узкий span верхней четверти
  - ♣ (трефа): два круглых лепестка → широкий span

## Устранение проблем

| Симптом | Причина | Решение |
|---------|---------|---------|
| Карты не читаются | Пресет не совпал с разрешением | Запусти `calibrate.py` |
| template-rate < 90% | Шаблоны из другого разрешения | Пересобери `collect_templates.py` |
| Масть всегда ♣ | Карты слишком маленькие | Уменьши `card_height_pct` на 1-2 |
| Банк = 0 | Регион не откалиброван | Перекалибруй Фазу 2 |
| Сервер недоступен | Неверный `server_url` | Скопируй URL из Replit |
| Постоянный FOLD | Неверный `card_height_pct` | Попробуй 8, 9, 10, 11 |

```
python/
├── poker_scanner.py        — основной сканер
├── calibrate.py            — интерактивная калибровка
├── collect_templates.py    — сбор шаблонов карт
├── load_preset.py          — загрузка готового пресета
├── card_utils.py           — общие функции: detect_suit, extract_card_region

├── config.example.json     — пример конфига
├── requirements.txt        — зависимости Python
├── presets/
│   ├── ton_poker_desktop_6max.json
│   └── ton_poker_6max.json
└── templates/              — 52 PNG шаблона карт (создаётся автоматически)
```
