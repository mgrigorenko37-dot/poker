# Poker Scanner — локальный Python агент

Захватывает экран, распознаёт карты, отправляет GTO-анализ в Telegram.  
Работает на Windows (тестировалось с World Poker Club на crazypandagames.com).

---

## Быстрый старт

### 1. Установи зависимости

```bash
pip install -r requirements.txt
```

> При первом запуске EasyOCR скачивает модель (~100 МБ) — нужен интернет.

### 2. Создай config.json

Скопируй `config.example.json` → `config.json` и поправь:

```json
{
  "server_url": "https://ТВОЙ_ДОМЕН.replit.dev/api/python/scan",
  "position": "BTN",
  "players": 6,
  "card_height_pct": 9
}
```

`server_url` — URL API-сервера. Найдёшь в настройках Replit-проекта (кнопка «Copy URL»).  
`card_height_pct` — высота карты в % от высоты экрана. Для большинства браузерных румов: **7–12**.

### 3. Откалибруй позиции карт

```bash
python calibrate.py
```

Нажми Enter → скрипт сделает скриншот экрана → откроется окно.  
Кликни по **центру** каждой карты в таком порядке:

| # | Что кликать |
|---|-------------|
| 1 | Твоя карта (левая) |
| 2 | Твоя карта (правая) |
| 3 | Борд — флоп левая |
| 4 | Борд — флоп средняя |
| 5 | Борд — флоп правая |
| 6 | Борд — тёрн *(Q — пропустить)* |
| 7 | Борд — ривер *(Q — пропустить)* |

Нажми **S** для сохранения. Калибровка записывается в `config.json`.

> ⚠️ Важно: разверни окно игры на весь экран ДО калибровки.  
> Если изменил размер окна — перекалибруй заново.

### 4. Запусти сканер

```bash
python poker_scanner.py
```

В терминале будет видно:
```
→ CALL  Win 24%  |  Ah Kd | 2c 7h Qd
→ FOLD  Win 8%   |  7c 2h |
→ RAISE  Win 67% |  As Ad | Kc 7h 2d
```

Одновременно в Telegram прилетит:
```
🟢 CALL
A♥ K♦  ·  2♣ 7♥ Q♦
Win 24%
```

## Настройка позиции и числа игроков

В `config.json`:

```json
"position": "BTN",   // UTG / MP / HJ / CO / BTN / SB / BB
"players": 6         // 2–9
```

Меняй прямо во время игры — сканер подхватит при следующем перезапуске.

## Устранение проблем

| Симптом | Причина | Решение |
|---------|---------|---------|
| Карты не читаются | Окно не на том же экране | Перекалибруй с игрой на основном мониторе |
| Постоянно FOLD на сильных руках | card_height_pct неверный | Попробуй 7, 8, 10 — смотри debug-превью |
| Дубли в терминале | OCR путает похожие карты | Уменьши scan fps или увеличь MIN_CONFIDENCE |
| Сервер недоступен | Неверный server_url | Скопируй URL из Replit (должен быть `.replit.dev`) |
