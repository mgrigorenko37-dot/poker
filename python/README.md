# Poker Scanner — локальный Python агент

Захватывает экран, распознаёт карты и суммы, отправляет GTO-анализ в Telegram.  
Работает на Windows (тестировалось с World Poker Club на crazypandagames.com).

---

## Быстрый старт

### 1. Установи зависимости

```bash
pip install -r requirements.txt
```

> При первом запуске EasyOCR скачивает модель (~100 МБ) — нужен интернет.

---

### 2. Создай config.json

Скопируй `config.example.json` → `config.json` и поправь:

```json
{
  "server_url": "https://ТВОЙ_ДОМЕН.replit.dev/api/python/scan",
  "position": "BTN",
  "players": 6,
  "card_height_pct": 9
}
```

`server_url` — URL API-сервера из Replit-проекта.  
`card_height_pct` — высота карты в % от высоты экрана. Обычно **7–12**.  
`position` / `players` — меняй вручную между раздачами (или автоматизируй позже).

---

### 3. Откалибруй позиции (карты + деньги)

```bash
python calibrate.py
```

**Фаза 1 — Карты** (клики):

| # | Что кликать |
|---|-------------|
| 1 | Твоя карта — левая |
| 2 | Твоя карта — правая |
| 3 | Борд — флоп левая |
| 4 | Борд — флоп средняя |
| 5 | Борд — флоп правая |
| 6 | Борд — тёрн *(Q = пропустить)* |
| 7 | Борд — ривер *(Q = пропустить)* |

Нажми **S** → переход к Фазе 2.

**Фаза 2 — Деньги** (прямоугольники):

| # | Что обводить |
|---|-------------|
| 1 | Цифра банка (pot) |
| 2 | Цифра колла/ставки *(Q = пропустить)* |

Зажми ЛКМ и обведи **только цифру**, не подпись.  
Нажми **S** для сохранения.

> ⚠️ Разверни окно игры на весь экран ДО калибровки.  
> При изменении размера окна — перекалибруй.

---

### 4. Запусти сканер

```bash
python poker_scanner.py
```

В терминале увидишь:
```
→ CALL  Win 24%  |  Ah Kd | 2c 7h Qd  банк=120  колл=30
→ FOLD  Win 8%   |  7c 2h |
→ RAISE  Win 67% |  As Ad | Kc 7h 2d  банк=45
```

В Telegram одновременно:
```
🟢 CALL
A♥ K♦  ·  2♣ 7♥ Q♦
Win 24%
```

## Клавиши в calibrate.py

| Клавиша | Действие |
|---------|---------|
| ЛКМ (клик) | Зафиксировать центр карты (Фаза 1) |
| ЛКМ (drag) | Нарисовать прямоугольник денег (Фаза 2) |
| Q | Пропустить борд4/5 или bet-регион |
| R | Начать текущую фазу заново |
| S | Сохранить и перейти дальше |
| ESC | Отмена |

---

## Устранение проблем

| Симптом | Причина | Решение |
|---------|---------|---------|
| Карты не читаются | Регионы не на картах | Включи `👁 Что видит OCR` в браузерном приложении или перекалибруй |
| Банк всегда 0 | Не откалиброван или фон мешает | Перекалибруй фазу 2, обведи только цифру |
| Банк читается неверно | Суффиксы K/M / пробелы | Скрипт обрабатывает K и M автоматически — проверь `card_height_pct` |
| Сервер недоступен | Неверный server_url | Скопируй URL из Replit |
| Постоянный FOLD | Неверный `card_height_pct` | Попробуй 7, 8, 10, 11 |
