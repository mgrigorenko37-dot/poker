# Poker Scanner — локальный Python агент

Захватывает экран, распознаёт карты и суммы, отправляет GTO-анализ в Telegram.  
Работает на Windows (тестировалось с World Poker Club на crazypandagames.com).

---

## Быстрый старт

### 1. Установи зависимости

```bash
pip install -r requirements.txt
```

> При первом запуске EasyOCR скачивает модель (~100 МБ).

---

### 2. Создай config.json

Скопируй `config.example.json` → `config.json`:

```json
{
  "server_url": "https://ТВОЙ_ДОМЕН.replit.dev/api/python/scan",
  "position": "BTN",
  "players": 6,
  "card_height_pct": 9
}
```

---

### 3. Калибровка (карты + деньги)

```bash
python calibrate.py
```

**Фаза 1 — Карты** (клики по центру):

| # | Что кликать |
|---|-------------|
| 1–2 | Твои карты (левая, правая) |
| 3–5 | Борд — флоп (3 карты) |
| 6 | Борд — тёрн *(Q = пропустить)* |
| 7 | Борд — ривер *(Q = пропустить)* |

Нажми **S** → Фаза 2.

**Фаза 2 — Деньги** (зажать и обвести):

| # | Что обводить |
|---|-------------|
| 1 | Цифра банка |
| 2 | Цифра колла *(Q = пропустить)* |

**S** = сохранить и выйти.

> ⚠️ Разверни игру на весь экран ДО калибровки. Изменил окно — перекалибруй.

---

### 4. Собери шаблоны карт (один раз, 2-3 раздачи)

```bash
python collect_templates.py
```

Просто играй в обычном режиме. Скрипт автоматически сохраняет шаблоны карт с уверенностью > 75%. Прогресс виден в терминале:

```
[██████████████████░░░░░░░░░░░░░░░░░░░░░░] 18/52
  💾 Ah  (уверенность 0.91)
  💾 Kd  (уверенность 0.88)
```

Когда все 52 карты собраны — скрипт сам останавливается.

> Шаблоны хранятся в папке `templates/`. Их нужно собрать только **один раз** для каждого рума/скина.

---

### 5. Запусти сканер

```bash
python poker_scanner.py
```

При старте выводится какой режим активен:

```
🃏 Template matching: загружено 52/52 шаблонов
💰 Банк: OCR
💰 Колл: OCR
Сканирование запущено (5 FPS). Ctrl+C для остановки.
```

В терминале по мере игры:

```
→ CALL  Win 24%  |  Ah Kd | 2c 7h Qd  банк=120  колл=30
→ FOLD  Win 8%   |  7c 2h |
→ RAISE  Win 67% |  As Ad | Kc 7h 2d  банк=45
```

В Telegram одновременно:

```
🟢 CALL
A♥ K♦  ·  2♣ 7♥ Q♦
Win 24%
```

## Как работает распознавание карт

| Режим | Когда | Скорость | Точность |
|-------|-------|----------|----------|
| **Template matching** | Шаблон для карты есть | ~2–5 мс | ~99% |
| **EasyOCR (fallback)** | Шаблон не найден | ~100–300 мс | ~85% |

Сканер автоматически использует template matching для карт у которых есть шаблон, и OCR для остальных. Когда все 52 шаблона собраны — EasyOCR для карт не запускается вообще (только для чисел банка/ставки).

Раз в ~12 секунд в терминале выводится статистика:
```
[stat] tmpl=47 ocr=0 miss=1  tmpl-rate=98%
```

## Устранение проблем

| Симптом | Причина | Решение |
|---------|---------|---------|
| Карты не читаются | Регионы не совпадают с экраном | Перекалибруй (размер окна изменился) |
| template-rate < 90% | Шаблоны из другого разрешения/скина | Пересобери через collect_templates.py |
| Банк всегда 0 | Не откалиброван или фон мешает | Перекалибруй Фазу 2, обведи только цифру |
| Банк неверный | Суффиксы K/M | Скрипт обрабатывает K/M/К/М автоматически |
| Сервер недоступен | Неверный server_url | Скопируй URL из Replit |
| Постоянный FOLD | Неверный card_height_pct | Попробуй 7, 8, 10, 11 |
